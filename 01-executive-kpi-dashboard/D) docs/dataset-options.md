# Dataset Options

## Ideal Dataset Type
Public sales or e-commerce data with enough detail to support KPI reporting, trend analysis, category analysis, and regional breakdowns.

## What the dataset should include
- Order date
- Revenue / sales amount
- Profit or cost data
- Product category
- Region or geography
- Customer or segment info if available

## Selection Criteria
- Public and non-sensitive
- Clean enough to work with
- Rich enough to create an executive dashboard
- Easy to explain during interviews
