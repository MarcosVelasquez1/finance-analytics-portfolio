# Driver-Based Forecast Model

This project will showcase a dynamic financial forecasting model built with public data and structured around key business drivers.

## Goal
Build a clean, professional forecast model that links operational assumptions to revenue, expenses, and profitability.

## Core Drivers
- Volume
- Price
- Growth rate
- Fixed costs
- Variable costs
- Scenario assumptions

## Tools
- Excel
- SQL
- Power BI

## Status
In progress
