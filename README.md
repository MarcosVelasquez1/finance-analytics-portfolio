# Finance & Analytics Portfolio

A curated portfolio of finance, FP&A, and analytics projects built using public data to demonstrate real-world business analysis, forecasting, reporting, and decision-support skills.

## What This Portfolio Demonstrates

- Executive-ready dashboard design
- Driver-based financial modeling
- Short-term cash flow forecasting
- Headcount and workforce planning
- SQL-based business analysis
- Strong presentation, structure, and business storytelling

## Tools Used

- Excel
- Power BI
- Tableau
- SQL
- GitHub

## Featured Projects

### 1. Executive KPI Dashboard
A polished executive dashboard focused on revenue, profitability, trends, and business performance drivers.

### 2. Driver-Based Forecast Model
A dynamic forecast model linking operational assumptions to revenue, expenses, and profitability.

### 3. 13-Week Cash Flow Model
A practical short-term cash flow model designed to monitor liquidity, weekly cash movement, and funding needs.

### 4. Headcount & Workforce Planning Model
A workforce planning model built to forecast hiring needs, payroll expense, and departmental growth.

### 5. SQL Analytics Pack
A structured SQL project demonstrating data cleaning, KPI logic, aggregation, and business performance analysis.

## Purpose

This portfolio is being built to showcase practical, visually polished, and decision-ready work product that reflects the standards expected in modern FP&A and analytics roles.
