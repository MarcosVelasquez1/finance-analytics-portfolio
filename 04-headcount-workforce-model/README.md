# Headcount & Workforce Planning Model

This project will showcase a structured workforce planning model used to forecast headcount, compensation, and department-level hiring impact.

## Goal
Build a planning model that connects hiring assumptions to payroll expense, team growth, and overall operating expense planning.

## Focus Areas
- Headcount by department
- Hiring plan by month
- Salary and compensation assumptions
- Open roles vs filled roles
- Workforce cost forecasting

## Tools
- Excel
- SQL
- Power BI

In progress
